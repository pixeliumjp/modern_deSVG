deSVG

  • Remove inline SVG bloat from your HTML document.
  • Style SVGs with CSS.
  • SVGs still accessible without JavaScript.
Download Docs Demo npm

What does it do?

deSVG takes the <img /> tags you supply. It then grabs, using AJAX, the raw SVG you've set in the "src" attribute and replaces that <img /> with the <svg /> it downloads.

Include the JavaScript file on your page

<script src="path/to/desvg.js"></script>

Kick off deSVG

window.addEventListener('load', function(){
 	// 1. selector for the <img /> tags to replace
 	// 2. whether to strip inline style tags from SVG paths
	deSVG('.replace-svg', true);
});

ノート

このライブラリを使用すると、HTTPリクエストが重複します（最初に画像の src をダウンロードするリクエストがあり、その後でSVGデータをfetchで取得するリクエストが発生します）。 ただし、開発は常にトレードオフの連続なので、このライブラリを使用することのメリットとデメリットを考慮するのはあなた次第です。時には、微小な最適化と、安定した開発ワークフローのどちらを選ぶかという問題になります。 あなたの判断を信じます。
